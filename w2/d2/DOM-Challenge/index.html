DOM Challenge

  1. Create a new project
  2. 3 files, 1 HTML, 1 CSS, 1 JS
  3. Open the html with live server or lite-server
  4. CSS:
    1. Set everything to box-sizing:border-box;
    2. Change the font (google font)
  5. HTML:
    1. Create an input with an id
    2. Declare a paragraph with an id
  6. JS:
    1. Initialize a constant for each element with an id created previously
    2. Set an event listener on the input (type of the event should be "input")
    3. Handle the listener with an event handler
    4. Access the input's value in your event handler
    5. Change the content of the paragraph with the input's value
